# Python Math and Data science 
## Basic setup
1. Python version 3.13 or more
2. Install basic library such as numpy, pandas, matplotlib etc. using pip or uv or poetry 
3. Use jupyter notebook for coding.
4. Store and use all the data in data_core folder. DO NOT PUSH THE DATA use .gitignore 
5. Add .gitignore properly
6. Use parquate file over CSV.
7. Manage requirements.txt for reproductivity and installation
